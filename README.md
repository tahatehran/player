# پلیر آنلاین ویدیو (Ththt.ir)

این مخزن نسخه‌ی فرانت‌اند یک پلیر آنلاین ویدیو است که با PHP، JavaScript و CSS پیاده‌سازی شده و برای پخش لینک‌های مستقیم ویدیو از دامنه‌های مشخص **Ththt.ir** طراحی شده است.

## فایل‌های اصلی

- `/home/runner/work/player/player/index.php` رابط کاربری اصلی، اعتبارسنجی لینک و ساختار صفحه
- `/home/runner/work/player/player/player.js` منطق کامل پلیر و کنترل‌های تعاملی
- `/home/runner/work/player/player/style.css` استایل اختصاصی رابط کاربری

## امکانات

- پخش/توقف، جلو/عقب ۱۰ ثانیه
- جابجایی فریم‌به‌فریم
- کنترل صدا و بی‌صدا
- تنظیم سرعت پخش
- چرخش ویدیو
- حالت تمام‌صفحه و Picture-in-Picture
- نوار پیشرفت با پشتیبانی Drag و Touch
- نمایش وضعیت پخش و پیام‌های مرکزی
- طراحی ریسپانسیو (موبایل و دسکتاپ)

## ورودی قابل قبول

پارامتر `u` در Query String باید یکی از پیشوندهای زیر را داشته باشد:

- `https://download.ththt.ir/file/`
- `https://online.ththt.ir/file/`

نمونه:

`index.php?u=https://download.ththt.ir/file/example.mp4`

## میانبرهای کیبورد

- `Space`: پخش/توقف
- `←` / `→`: عقب/جلو ۱۰ ثانیه
- `↑` / `↓`: کم‌وزیاد کردن صدا
- `M`: بی‌صدا
- `F`: تمام‌صفحه
- `P`: Picture-in-Picture
- `Q` / `E`: چرخش چپ/راست
- `,` / `.`: فریم قبلی/بعدی
- `0` تا `9`: پرش به درصدهای ویدیو

## اجرا

این پروژه Build Step ندارد. برای اجرا کافی است مخزن را روی یک وب‌سرور PHP قرار دهید و فایل `index.php` را باز کنید.
